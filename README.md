# 学习DOTA 2的技术与战术

> 本库为我自己学习DOTA2的游戏笔记, 主要为基本概念的说明, 英雄和道具的理解, 游戏玩法与策略. 

~~争取早日上统帅~~

DOTA2, 策略竞技推塔游戏的唯一真神. 从基本的分工上将, 每个玩家被分为1, 2, 3, 4, 5号位中的一位, 在不同的游戏阶段担负起不同的责任. 从游戏进程上将, 分为前期, 中期和后期. 不同的游戏阶段将决定不同的进攻防守策略. 而游戏的最终目的, 就是攻城略地, 夺取对方的防御塔和发育空间. 最后进攻对方的遗迹, 获取胜利. 

而DOTA2的游戏乐趣就在于, 从战斗号角响起前的一分三十秒, 到一方遗迹的最后一滴血的过程中, 这个游戏提供了无数的可能性. 从英雄的选择与位置分配, 针对性的进攻和防守眼位, 大哥刷钱路线的规划和刷打结合的时机, 酱油在战斗中和战斗外的运营牵扯. 在令人叹为观止的复杂英雄与技能的组合中, 蕴含着深思熟虑的决策与老谋深算的经验. 细节体现在数值上的最优解, 谋略体现在对地图的解读和预测. 唯有那一支技术最熟练, 战术最灵活, 心智最成熟的队伍, 才能在无穷的竞技可能中, 洞察破局的最佳决策. 

DOTA2中英雄和道具组合的策略深度和乐趣性之间的平衡让这款游戏成为了一款真正独特的游戏体验, 使我不断的学习, 成长并追求更高的竞技水平. 

## 游戏过程

游戏过程指在不同的事件做不同的事. 最粗糙的划分为:

- 前期: 对线, 获取最基础的经验和经济.
- 中期: 夺取目标点, 运营地图视野. 进一步扩展优势或者是弥补前期劣势. 
- 后期: 核心拿到关键装备, 开始高地进攻战. 

第一阶段为中期, 也就是对线期. 各个英雄根据自己的位置在自己的线上展开正反补的争夺, 同时寻找击杀对面英雄的机会,获取最基础的经验和经济. 一般会在游戏的0~10分钟. 十分钟之后, 发育正常的英雄都具备了最基础的装备, 同时到达6级, 获得自己的第一个大招, 英雄的能力有了一个质的飞跃. 游戏进入下一个阶段. 

第二阶段为中期, 是具备了一定经济和等级的英雄开始在整个地图上大展拳脚的时间. 前文提到, **DOTA2本质是一款攻城略地的游戏**, 通过推倒防御塔和占领视野等方式获得更大的地图权, 进而获得更多的地图资源, 为胜利铺路. 而游戏的中期, 就是英雄们开疆拔寨的时间. 这段时间的游戏主要目标是推掉对方三路尽可能多的塔. 而为了达到这个目标, 往往需要需要很多小目标的辅助: 如击杀对面英雄, 让对面无力防守防御塔的进攻; 而为了杀掉对面英雄, 往往中单会夺取双分钟刷新的神符, 利用神符带来的power surge创造击杀的机会. 又或者是为了获得不朽之守护(所带来的五分钟强势期), 围绕着肉山坑位开始团战. 中期的一切本质目标都是为了更大的地图权, 因为更大的地图权意味着更多的资源, 等价于更大的获胜几率. 

第三阶段为后期, 是一方的核心英雄取得了关键的装备, 已经具备足以威胁高地防御塔的强大能力, 开始向另一方遗迹展开最后攻势的阶段. 前期的对线, 中期的运营, 都是为了此时双方英雄互相决战而堆积筹码. 这个阶段最大的游戏特点就是英雄的死亡将极大的倾斜胜利的天平, 无论你是优势方还是劣势方! 因为此时的英雄等级较高, 死亡后需要一分多钟的复活时间. 若毫无防备被对方轻取, 则场面上人多的一方将具备极大的优势甚至是一波推平. 此时也是整个游戏的高潮节点: 遗迹守卫战往往是双方十人全部的战力和智力的最后斗争, 核心英雄具备了高额的输出能力, 辅助英雄也往往具备关键的功能性技能与道具. DOTA2最精彩的翻盘, 往往也发生在背水一战的绝境之中. 

三个阶段没有明确的时间节点划分, 而是由一方英雄累计的经济和等级决定的: 前期英雄都较为弱小, 无法有效的威胁对方防御塔, 因此分路补刀发育是明智的选择. 中期英雄初步成型, 可以进行有效的团战和扫图, 因此围剿对方地图资源是首要任务. 后期英雄能够在顶住对面高地的防守加成强取敌方要地, 所以也是发起最后进攻的好时机. 至此, 明确指出决定游戏阶段的唯一因素就是一方资源的多少, 所以整个游戏的本质就是要做到能比对面更快获取地图资源. 而游戏阶段是跟着更优势的一方走的, 当对方提前达累计到进行到中期的资源时, 游戏中期就已经开始了, 不会还等劣势一方慢慢对线的. 相对应的, 劣势一方的资源就被进一步压缩, 直到优势方累积到足够威胁到遗迹的资源时, 高地攻防战就开始了--无论另一方身上带的是魔棒还是狂战斧. 

### 前期: 对线的关键是从对方手上抢夺正反补

前文提到整个游戏的唯一本质就是争夺资源, 这样说或许稍显模糊. 那么在对线期, 成功的唯一衡量就是获取比对面更多的正反补. 因为**正反补 = 经济+经验**. 开局良好的装备和健康的等级是顺利进入游戏下一阶段的必要条件, 甚至可以说良好的对线是游戏获胜的一半. 所以获取正反补, 是对线期的关键任务. 

这就涉及到对线期的关键: **如何从对方手中抢走正反补?**

首先是基本功: 对英雄抬手的熟悉, 对小兵血量的计算, 能够在空补的情况下不漏刀. 一般来说, 力量英雄拥有较高的初始攻击力, 小兵的补刀血线较高. 敏捷英雄的抬手速度快, 可以比较快的攻击到小兵. 智力英雄就自求多福吧. 补塔刀时, 要数塔对小兵的攻击次数, 一级塔对近战小兵攻击六下, 要提前垫一刀然后再补; 对远程小兵攻击三下, 可以三下后直接补. 工程车不太好补, 我一般随缘. 

对线时一般不会有完全空补的时机, 往往地方英雄会来骚扰你的补刀. 这是就是对线期的主要游玩内容了: 对线压制. 当你压制到对面时, 你就有更好的补刀环境. 如何判断是否压制到了对面? 最简单是看对面的血量蓝量是不是比你少. 对面血少蓝少, 线上拿你没办法, 自然无法和你抢补刀. 

理解了对线的目标就是压制对方获取正反补, 如何实现这个目标就是对线期的关键. 对这个目标执行更好的一方, 往往就是这条线线优(十分钟经济经验总和高于对面)的一方. 实现这个目标的基本方式包括对拼耗血和控线拉野. 

**对拼耗血**讲的是如何打少对方的血量和蓝量. 这就涉及对线期的对拼与消耗了. 在均势对线中, 一般英雄简单互殴是换血, 对方的状态下去了你的状态也下去了, 等于还是均势. 这个时候, 倾动对戏平衡的砝码就是: 技能与道具. 

DOTA2 中的技能与道具在对线期是非常重要并且珍稀的东西. **技能**消耗蓝量, 除了少数的耗蓝好CD短的技能, 大部分技能在对线期的使用次数都非常有限, 因此, 非常有必要最优化技能释放后取得的压制效果. 如何最大化技能施放的效果? 最重要的是不要乱丢技能, 手中捏着的技能就是一个对拼的筹码, 可以撬动对线天平的砝码. 其次, 要对技能的效果心里有数, 技能一般的效果包括伤害, 减速, 眩晕, 位移等等. 有伤害的技能, 就要算好伤害和对面的血量, 当对面的血量低于"斩杀线"的时候, 就可以视情况释放技能. 有减速的技能, 就方便我方追着打一顿, 所以当对方远离防御塔和队友的时候追击就最方便使用, 眩晕要用来打断对方重要的技能释放, 比如读条技能和TP. 位移技能则可以将自己置于有利位置, 用于逃生或追击. 

**道具**需要金钱购买, 消耗经济. 在对线期中, 使用的道具主要包括吃树, 小蓝, 大药, 芒果, 仙灵之火, 榴弹, 树枝, 魂泪等. 这些一般称为补给, 用于恢复自己的状态, 保持这条线不被完全压制住. 记住, 自己的状态好就等于对面的状态差, 对面的状态差就等于这条线更好打. 若是辅助位置, 尤其要注重补给道具的购买, 消耗自己的经济, 为大哥创造更好的对线环境, 是这条线打好的关键. 一条线有没有线优, 看的是大哥的经济. 辅助的前期的经济, 其实无所谓的. 

> 技能施放和道具使用是随战场情况有机应变使用的, 心中一定要对自己和地方的对线筹码心中有数. 

**控线拉野**是除了技能和道具外影响对线平衡的另一个重要因素. 首先讲解一下对线期的位置因素. 以天辉方的优势路(天辉方优势路为下路, 夜魇方优势路为上路)为例, 对线期的位置一般发生在天辉方一塔前至夜魇方一塔前, 当双方的兵线在不受干扰的情况下交汇时, 交汇点会靠近天辉方一塔前. 而兵线的交汇点就是对线时的正面战场, 对线四人会围绕着交汇点进行对线. 此时设想, 当天辉方发起进攻时, 夜魇方开始逃离战场时, 夜魇方要一直撤离至较远的夜魇一塔才算安全. 回塔的路上会被天辉方一直追着打, 会被消耗大量的血量. 而当夜魇方发起进攻时, 天辉方可以很快撤回防御塔下, 享受防御塔的防守加成(回血速度和护甲). 夜魇方将没有很多的进攻空间. 因此, 当兵线交汇的位置越靠近我方防御塔时, 我们的对线会越有优势. 

所以, 这里涉及到如何将兵线尽量保持在我方塔前的问题. 讲解一下兵线为什么会推出去, 当一波兵线, 假定三个近战兵和一个远程兵和对方同等兵力相交时, 双方小兵相互消耗, 都阵亡时还会停留在原地, 若一方的英雄攻击了对面的小兵, 就会让我方小兵更有优势, 消耗完对方小兵之后还有余量, 兵线就会继续往前推进了. *值得一提的是, 远程小兵攻击高, 兵线的自平衡往往是哪一方的远程小兵多决定的.* 所以, 要想推进兵线, 就要消耗对方小兵, 要想拉回兵线, 就要消耗自己的小兵, 若要保持兵线在某个位置不动, 则需要保持两边的小兵数量相同. 这是控制兵线的原理. 

了解了为什么要控制兵线和兵线移动的原理, 接下来就是分别说明控线和推进的方法了. 首先, 兵线相互消耗的时候不要傻站在兵线附近攻击敌方小兵, 会推线的同时将你自己置于危险的位置(对面英雄攻击你时对面小兵也会开始打你). 只要补刀最后一刀, 即可在不推进兵线的情况下获取经验和金钱. 控线时要保持双方小兵数量一致同时类型一致, 有几个天辉近战兵就有几个夜魇近战兵, 有几个天辉远程兵就有几个夜魇远程兵. 数量对不上时, 赶紧攻击或者反补较多一方的小兵, 这样下一波兵线过来时, 就还会交汇在相同的位置, 就完成了一波控线. 若兵线被对方控制在对方塔前你补刀非常难受, 那么需要拉己方的野怪, 使用野怪消耗完自己的一波兵线, 拉回兵线的同时让对面吃不到一波兵的经验和金钱. 

拉野是对线中的一个关键, 在这里用一个自然段详细的说明. 中立生物(野怪)是存在于地图上独立于天辉和夜魇的怪物, 会对双方都产生仇恨. 利用这个机制, 可以让中立生物和自己的小兵交战, 互相消耗. 具体的方法是30s产生一波小兵, 等小兵走到野怪营地附近, 攻击一下野怪然后往兵线的方向上走, 等小兵看见了野怪, 就会主动和野怪交战, 直到一方被消灭. 具体拉野的时间会随地图不同位置的营地而变, 可以参考dota plus给出的提示时间. 一般对线期有四个营地可以用于拉野: 天辉方下路一塔左上角的小野营地, 天辉方上路一塔右上角靠近夜魇的大野营地, 夜魇方上路一塔右下角的小野营地, 夜魇方下路一塔左上角靠近天辉的大野营地. 具体的位置不用记住, 只需要知道优势路会有一个小野供你拉, 劣势路会有一个大野供你拉. 为什么会分大野和小野呢? 野怪的强度分为小野, 中野和大野, 更强的大野也会有更高的经济经验回报. 拉野时要注意, 一波小野无法消耗完兵线, 甚至会让兵线更加推出去. 大野可以一波消耗完一整波兵线. 所以, 拉小野时, 需要囤双野, 两波小野就可以消耗完一整波兵线, 让线拉回自己塔前. 

> 什么是囤野? DOTA2有个继承于war3的机制: 当野怪营地在整数分钟时没有任何单位时, 就会刷新出一波新的野怪. 所以, 当有之前的野怪(或者其他任何单位)在整数分钟时呆在了刷野范围内(可以使用dota plus的野怪刷新范围查看, 游戏中按下alt键即可), 就不会有新的野怪产生. 所以若我们在大约55s的时间攻击一下野怪, 然后离开营地方向, 野怪的仇恨机制就会驱使野怪追着我们打一小段距离. 这时候野怪离开了营地, 当时间转到整数分钟时, 又会刷新出一波新的野怪, 这样两拨野怪会占同一个坑, 就完成了一次堆野. 小野营地拉野时, 一定要提前囤双野后再拉, 这样才能保住清掉一波兵线. 

> 囤野的另一个作用是增加我方地图上的累计资源. 上文中提到每分钟都会刷新一次野怪, 若营地中有单位则不会刷新. 其实每次重新刷野都是游戏机制补充了地图上的资源, 若空的野怪营地多, 那么补充的就多. 所以, 当我们囤一波野怪时, 就是凭空增加了我方待获取的资源量. 所以囤野是非常重要的游戏机制, 不光影响对线, 也影响对线期后的发育速度. 中期打钱时, 尽可能囤野打多野, 这样可以提升获取资源的速度. 

控线讲完了, 那什么时候推线呢? 推线的作用有很多, 当线被推入对方塔内时, 对方只能缩在塔内清兵, 同时兵线会被防御塔攻击, 所以推线后反而可以把兵线借助对方防御塔推回来. 另外推线时一般我方小兵多于对方小兵, 这也是一个进攻的好机会, 所以可以借助推线杀人. 相比控线, 推线就很简单了, 一般使用AOE技能快速清掉小兵即可, 所以很多强势中单英雄往往都有清兵推线的技能. 要注意推线往往是一个利弊权衡的过程, 要是推线执行差了, 很可能起到反作用. 

对线时有可能会遇到线霸级别的英雄, 甚至连兵线的经验都吃不到, 一上兵线就有可能死. 这个时候, 再和对面打消耗战是非常不明智的. 此时一般要怂一点, 多绕树林找找机会, 多运补给保证自己的状态. 除此之外, 还有两件事可以弥补对线期的劣势: 囤己方的野怪稍后补发育, 或者选择拉线. 拉线是指劣势路一方绕到对面一塔后, 提前仇恨兵线将兵线带到安全位置再清掉. 这样可以让自己吃到一点兵线的经济和金钱. 但是要注意, 这种方法会深入敌后, 稍有不慎就会留给对面非常大的击杀机会. 移速慢的英雄一定不能去拉线. 

消耗, 控线, 拉野, 推线, 拉线都讲完了. 其实有个重要概念还没有解释, 就是兵线仇恨机制. 仇恨机制是上述操作的本质机理, 之所以存在对线的各种操作, 就是使用了不同的方式利用了小兵的仇恨机制. 仇恨机制的表现非常简单, 在近战小兵的一定范围内对敌方英雄发起攻击指令. 仇恨最简单最频繁的用法, 就是拉兵. 站在兵线相交的位置, 拉一下小兵的仇恨, 就会让几个近战小兵开始追着你打, 然后你把三个近战兵拉到自家远程兵的位置, 就会让三个敌方近战小兵围殴自家远程兵. 这样就可以让兵线拉回来一点点, 同时确保自己的远程兵消掉, 继续控线. 这个技巧是对线中最常使用的技巧, 用于远离对方进攻线安全补兵. 

> 防御塔, 中立生物也有自己的仇恨机制. 具体可以参考ref: https://dota.huijiwiki.com/wiki/%E4%BB%87%E6%81%A8

在7.34大展宏图更新后, 地图上新增了大量的时间节点目标, 包括莲花, 经验符以及双生之门. 莲花是3分钟倍数在边路刷新的消耗品, 瞬间恢复一定的血量和蓝量. 这种瞬间恢复的道具, 是对拼的时候比对面更能消耗的筹码, 所以一般不要拿到就吃了. 前文提到, 对线压制就是要比对面有更多的状态, 所以对线期的莲花也非常重要, 一定要去3分钟和6分钟时去抢到. 7分钟的经验符会刷新在劣势路二塔旁. 这个经验符是非常重要的前期资源, 辅助吃到了基本上能比对面更快拥有大招, 大哥吃到了能有更强的战斗力, 所以也是兵家必争之地. 对线期一般为十分钟, 现在的游戏节奏基本固定, 每分钟都有要做的事情, 所以对线期要对时间非常敏感. 

双生之门增加了游走的灵活性, 对线期能灵活运用双生门的一方往往能在对线期期间打出巨大优势. 我不会用, 所以我就不讲了, 没事多切切屏, 多看小地图, 多读信息, 是非常重要的习惯. 我不太会, 一般随缘. 

### 中期: 攻城略地, 侵占地图

对线期结束后, 游戏会来到中期. 中期的英雄有了围绕着地图的新的目标: 中立野怪资源和敌方防御塔. 前期的英雄往往不能有效的获取这两种资源, 但是具备一定战斗力的中期英雄可以和对面开始争夺这两种资源了. 

中立野怪等价于击杀者的经验和金钱. 核心英雄之所以能够成为核心英雄, 本质就是其能够快速获得包括中立野怪在内的各种资源, 迅速成长, 并在后期能将资源转换成极强的战斗力. 所以地图上的中立野怪往往是由核心英雄去获取的, 这也就是一般俗称的"刷钱". 

一个英雄的刷钱能力由三个方面决定: 从一个野点移动到下一个野点所需的时间, 清掉野点所需要的时间, 打野的消耗. 当英雄拥有高移速或者位移技能时, 其符合第一个条件; 当英雄具备AOE伤害时, 其符合第二个条件; 当英雄具备吸血或者高恢复时, 满足第三个条件. 使用典型刷钱英雄敌法师说明, 敌法师+狂战斧(必出)的配置, 敌法的闪烁满足条件一, 狂战斧提供的溅射满足条件二, 狂战斧提供的血蓝恢复满足条件三. 所以敌法师是刷钱非常快的英雄. 另外, 敌法的快速攻击间隔(1.4)决定了其后期不俗的攻速, 配合其被动可以造成大量伤害. 所以敌法师是毫无疑问的后期英雄. 多数核心英雄往往具备刷钱能力的一到两点, 刷钱方法和效率的不同, 团战切入输出方式的不同, 技能定位的不同, 这些就是各个核心英雄的特色所在. 

不同的野点对应的资源不同, 刷野优先级: 远古野>一波兵线>大野>中野>小野. 从这个优先级可以看出, 劣势路一方的资源质量是最高的(两个远古点+一个大野+兵线+经验符), 优势路一方资源更多(小吃街+主野区大量资源), 一般来说, 一分钟的刷新周期只够清掉地图上的一部分资源, 所以选择哪块区域刷钱, 要结合局势和英雄特色决定. 

中期随着防御塔的倒下, 兵线的位置将不再限制于一塔之间. 当我方兵线的位置缩在自己半区时, 我方视野受限, 活动范围受限; 

野怪和兵线资源虽然相对容易获得, 但是其对局势并没有任何实质性的改变. 真正重要的地图争夺点, 是敌方的防御塔. 

防御塔直接决定了一个队伍在一方的地图占领权. 当防御塔存在时, 防守防可以获得视野(包括真视视野), 护甲和回血, 最重要的是友方队友的迅速TP支援. 当防御塔不存在时, 兵线的压力会更大, 视野被进一步压制, 被抓也无法及时支援. 因此, 要想压缩对方的地图资源, 最直接的方法就是进攻拿塔.
